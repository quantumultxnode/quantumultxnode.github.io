一、小火箭简介：iOS用户的自由之门

“小火箭”并不是官方名称，而是Shadowrocket这款iOS端代理应用的中文昵称。作为一款功能齐全的网络代理客户端，小火箭支持的协议非常全面，包括：

  • Shadowsocks（SS）

  • ShadowsocksR（SSR）

  • Vmess / VLESS

  • Trojan

  • HTTP / HTTPS

  • SOCKS5

这意味着只要拥有正确的节点配置，你几乎可以实现对全球任意网络的访问。而小火箭最被用户津津乐道的地方在于：强大的规则系统、稳定的连接性能以及对V2Ray协议的良好支持。

二、Vmess协议概述：为何越来越多人选择它？

在V2Ray项目中，Vmess协议的设计目标就是打破传统Shadowsocks在深度检测中的弱点，提供更安全、抗干扰的代理能力。它具备以下几个优势：

  • 数据加密与认证：相比Shadowsocks，Vmess的数据包自带认证机制，可有效防止流量重放与伪装攻击。

  • 多种传输层支持：可通过TCP、WebSocket、HTTP/2等方式传输，甚至搭配TLS混淆，进一步增强隐匿性。

  • 配置灵活性高：支持更多参数自定义，比如UUID身份识别、动态端口、Header伪装等。

这些特点让Vmess成为在高干扰环境下依然能保持畅通连接的理想选择。

三、开始之前：你需要准备什么？

在正式配置Vmess前，你需要准备以下内容：

  1. 已安装的小火箭客户端（来自海外Apple ID的App Store购买，或越狱方式安装）

  2. 一个有效的Vmess节点链接，一般由服务提供商提供，格式通常以 vmess:// 开头，或通过JSON数据提供。

  3. 稳定的网络环境，建议使用Wi-Fi或4G，保证设置过程不中断。

步骤二：添加Vmess节点到小火箭

方法一：自动导入

  1. 复制vmess链接

  2. 打开小火箭，点击右上角“+”

  3. 选择“从剪贴板导入”

  4. 小火箭会自动识别并生成配置，点击“完成”即可添加成功

方法二：手动输入

适用于手动获取节点信息时，步骤如下：

  1. 打开小火箭，点击右上角“+”

  2. 协议选择“Vmess”

  3. 填入以下关键信息：

    • 地址（Address）：服务器域名或IP

    • 端口（Port）：服务端口，常见如443

    • 用户ID（UUID）：服务商提供的唯一用户识别码

    • 额外ID（alterId）：部分服务设定为0，已废弃可留空

    • 加密方式：推荐使用auto或aes-128-gcm

    • 传输协议：TCP或WebSocket（ws）

    • TLS：如节点支持，务必启用以提高隐私

    • 伪装路径（path）：如使用ws协议，则需填写路径，如/path

    • Host：伪装域名或cdn域名

  4. 配置完成后，点击“保存”

步骤三：启动并测试连接

  • 返回小火箭首页

  • 选择刚添加的Vmess配置

  • 点击开启VPN开关

  • 若状态栏出现VPN图标，说明连接成功

可以通过访问被屏蔽的网站（如YouTube、Google）测试效果。

六、常见问题解答

Q1：小火箭打不开或下载不到怎么办？

A1：如App Store中搜索不到Shadowrocket，需切换Apple ID至海外（如美国、香港）账号才能下载。

Q2：连接不上怎么办？

A2：建议按以下顺序排查：

  • 检查网络是否通畅

  • 检查UUID、端口等是否填写错误

  • 切换协议类型（TCP ⇄ WS）

  • 尝试不同节点

Q3：Vmess链接失效了？

A3：Vmess服务本质是点对点代理，一旦服务商更换配置或服务器宕机，链接就会失效。应及时联系服务方更新配置。

Q4：使用Vmess安全吗？

A4：Vmess相较SS协议在协议伪装和传输加密方面更强，若搭配TLS/WS使用，隐私保护和抗干扰能力更高。

FAQ

Mellow TUN 模式适合哪些应用？
TUN 模式拦截系统 IP 层流量，实现全局代理。适合游戏、视频、浏览器等全局应用，保证所有流量走节点，提高网络稳定性和速度，比普通 SOCKS5 代理覆盖范围更广。
VLESS + XTLS 在移动网络的优势？
XTLS 模式减少握手次数和加密开销，适合高延迟或不稳定的移动网络。结合策略组和多节点切换，可提升网页浏览、视频播放和游戏的连接稳定性和流畅度。
如何给 Clash 或 Sing-box 添加本地广告屏蔽规则？
可导入公共广告域名列表（如 ads、trackers）并将其指向 `REJECT` 或 `DIRECT`（根据需求）；还可以使用 rule-provider 自动更新广告规则，并结合 hosts 或 DNS 层面进行联合拦截以提升屏蔽效果。
Nekoray 节点订阅如何实现自动更新？
用户可添加订阅 URL，Nekoray 会定期自动下载并更新节点列表。保持节点最新可减少手动维护，提高节点可用性和网络连接稳定性，确保科学上网体验顺畅。
Shadowrocket 的测速功能如何优化节点选择？
Shadowrocket 可测量节点延迟和丢包率，并根据结果排序策略组节点。延迟低的节点会优先使用，保证网页浏览、视频和游戏的速度和稳定性，提高整体网络体验。
Nekoray 能否同时运行多个代理？
Nekoray 支持多配置运行。用户可在“多配置模式”下同时开启不同协议或节点的代理，程序会自动创建多个监听端口，方便多应用或分流测试使用。
Netch 如何区分全局代理和规则代理？
全局代理会将所有流量通过节点转发，而规则代理仅对指定 IP、端口或应用生效。用户可根据需求选择代理模式，实现流量优化和不同应用的最佳网络体验。
SSR 的混淆和协议设置如何选择才能绕过封锁？
选择模拟 HTTPS 的混淆方式如 tls1.2_ticket_auth，并搭配 AEAD 或协议插件，可以有效隐藏流量特征，降低被封锁风险。不同网络环境需测试选择最优组合。
SSR 的协议参数如何提高抗封锁能力？
SSR 的协议参数可以模拟不同流量特征，使加密流量更像普通 HTTP 或 HTTPS。合理选择协议类型和混淆参数，可以有效降低被封锁的风险，提高节点长期可用性。
WinXray 是否支持多订阅源？
支持。用户可导入多个订阅 URL，每个订阅源独立管理。通过切换订阅源或合并策略组，可灵活管理节点，实现不同地区或用途的线路分配。
